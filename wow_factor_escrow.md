## 🔒 Setting Up the On-Chain Escrow Smart Contract

The Solidity code executing the agreement is located at [contracts/MonAgent.sol](file:///c:/monad/contracts/MonAgent.sol). Here is how it functions:

### 1. Escrow Agreement Struct
The contract stores active agreements in a mapping keyed by a unique `bytes32` identifier (generated by hashing the agent address and current timestamp):
```solidity
struct Agreement {
    address payable client;
    address payable agent;
    uint256 amount;
    bool isCompleted;
    bool isReleased;
    bool isRefunded;
}

mapping(bytes32 => Agreement) public agreements;
```

### 2. Lock Funds in Escrow
When the client accepts the candidate, the frontend initiates a deposit. The client calls the `createAgreement` function, sending the exact bid amount as message value (`msg.value`). The contract locks the `MON` tokens in its balance:
```solidity
function createAgreement(bytes32 agreementId, address payable agent) external payable {
    require(msg.value > 0, "Deposit must be greater than 0");
    require(agreements[agreementId].client == address(0), "Agreement ID already exists");

    agreements[agreementId] = Agreement({
        client: payable(msg.sender),
        agent: agent,
        amount: msg.value,
        isCompleted: false,
        isReleased: false,
        isRefunded: false
    });

    emit AgreementCreated(agreementId, msg.sender, agent, msg.value);
}
```

### 3. Verify & Release Funds
Once the agent completes the work and the live debugger console finishes compiling, the client clicks **Release Escrow Payment**. The frontend calls `releasePayment`, verifying that the sender is the client and transferring the locked funds to the agent:
```solidity
function releasePayment(bytes32 agreementId) external {
    Agreement storage agreement = agreements[agreementId];
    require(msg.sender == agreement.client, "Only client can release payment");
    require(!agreement.isReleased, "Payment already released");
    require(!agreement.isRefunded, "Agreement already refunded");

    agreement.isReleased = true;
    agreement.isCompleted = true;

    uint256 paymentAmount = agreement.amount;
    agreement.amount = 0; // Prevent reentrancy

    agreement.agent.transfer(paymentAmount);

    emit PaymentReleased(agreementId, paymentAmount);
}
```

### 4. Refund Agreements
If the agent fails verification tests, the client can request a refund. The funds are sent back to the client:
```solidity
function refundClient(bytes32 agreementId) external {
    Agreement storage agreement = agreements[agreementId];
    require(msg.sender == agreement.client, "Only client can request refund");
    require(!agreement.isReleased, "Payment already released");
    require(!agreement.isRefunded, "Already refunded");

    agreement.isRefunded = true;

    uint256 refundAmount = agreement.amount;
    agreement.amount = 0; // Prevent reentrancy

    agreement.client.transfer(refundAmount);

    emit RefundExecuted(agreementId, refundAmount);
}
```
Deploying this contract to the Monad network secures the client-to-agent operations, ensuring trustless execution where agents are only paid upon delivering verifiable code.
